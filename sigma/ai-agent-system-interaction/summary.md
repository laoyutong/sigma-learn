## 核心知识点回顾
1. **物理边界**: 明确了 LLM（大脑，仅处理文本）、Host（手脚，执行代码）和 Target（被操作对象）的严格分离。
2. **工具协议**: 掌握了 Host 必须将工具的 JSON Schema 发送给 LLM，LLM 才能返回正确的函数调用 JSON。
3. **MCP 架构**: 理解了 Model Context Protocol 如何通过 Client-Server 模式解决 N×M 集成地狱，实现工具的动态发现和解耦。
4. **权限管理**: 认识到静态作用域（白名单）和动态授权（Human-in-the-loop 二次确认）在防止 Agent 暴走中的重要性。
5. **沙箱隔离**: 深入了解了 Docker、WASM 和操作系统级沙箱（seccomp）如何作为最后一道防线，在底层拦截恶意的系统调用。
