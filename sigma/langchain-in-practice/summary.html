本次学习的关键洞察
▸Runnable 协议是 LangChain 一切组合能力的基础——.pipe() 之所以能随意串联，是因为所有组件实现了统一接口
▸prompt.invoke() 只格式化消息，不调用模型——必须 .pipe(model) 才触发 API 调用
▸.describe() 只影响发给模型的 JSON Schema（元数据），不影响 TypeScript 类型也不设默认值
▸AgentExecutor 的 while 循环轮数 = 工具调用次数 + 1（最后一轮是直接回答）
▸RAG 建库和查询必须用同一个 Embedding 模型——不同模型的向量空间不兼容
▸AgentExecutor = 固定线性循环；LangGraph = 任意形状的有向图，两者适用场景完全不同
